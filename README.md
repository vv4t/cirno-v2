# cirno-v2

## Requirements

- SDL2

## Build Instructions

> make

## Examples

### SDL

```
When running windowed scripts add the -w flag
./cirno -w demo_sdl/wave.9c
./cirno -w demo_sdl/bounce.9c
./cirno -w demo_sdl/planet.9c
```

### CLI

Input
``` 
./cirno demo_cli/input.9c
[OUTPUT]
Name: [enter your name]
Hi [name]!
```
